## 2. 國產晶片加分：達成方式與佐證（重點）

**加分定義**：採用「基於國產晶片開發平台」實現**近端（使用者端）系統原型演示**。符合下列之一即可：自主研發晶片／核心矽智財／開源架構改良。

**本專案如何達成**：

- **核心晶片為國產**：開發板核心 SoC 為 **MediaTek Genio 520**，聯發科為我國依法登記之企業、自主研發並擁有該晶片 IP，**符合「自主研發晶片」條件**。
- **近端原型真實運用晶片算力**：透過 **NeuroPilot Public（免 NDA）** 將 ASR/TTS 轉 `.tflite` INT8，經 **Neuron Delegate 上 Genio 520 NPU** 硬體加速；LLM 以 llama.cpp 跑其 Cortex-A78 CPU。即「使用者端原型」確實在國產晶片上運算。
- **與 LLM 選型無關**：邊緣用 Qwen2.5-1.5B 或任何模型皆不影響加分——加分綁定的是**晶片平台**。

**現場佐證清單**：

1. 實機展示 Genio 520 開發板運行原型（近端演示）。
2. 晶片/開發板來源與規格文件（聯發科官方頁、開發板型號）。
3. on-device 推論證據：NPU 加速前後延遲對照、`.tflite` 部署截圖/log。
4. 系統架構圖標示「近端＝Genio 520」。

## 3. 現場 Demo 分鏡腳本（約 4 分鐘）

> 目標：在最短時間內證明「真的能離線在國產晶片上跑」與「教師閉環」。**全程備妥預錄影片，硬體異常即切。**

**鏡頭 0（15s）痛點鉤子**：一句話點出偏鄉英語口說 + 斷網 + 師資不足痛點，帶一個實證數字。

**鏡頭 1（60s）實機離線對話**：拿起企鵝玩偶，學童說一句中英夾雜（如「I want to eat 蘋果」）。玩偶用台灣口音鷹架帶讀回應（「跟我說：I want to eat an apple」）。強調**這一切跑在手上這台 Genio 520，沒連網**。腹部 LED 呼吸燈律動。

**鏡頭 2（40s）斷網橋段（記憶點）**：主持人當場開飛航模式/拔網路，再對話一次——**玩偶照常即時回應**。點出「偏鄉沒網路也不中斷，這是普惠的關鍵」。

**鏡頭 3（30s）國產晶片佐證**：切一張投影片：Genio 520＝聯發科自主研發晶片；ASR/TTS 經 NeuroPilot Public 上 NPU；展示 on-device 延遲/記憶體實測數字。

**鏡頭 4（50s）教師閉環**：連上網，剛才的互動資料同步到教師儀表板；點開學生，Hermes Agent（接 Bedrock）產出的四維評分、弱項、差異化教學建議與長期趨勢一覽。強調 agent **持久記憶**能長期追蹤每位學生。

**鏡頭 5（25s）願景收束**：無螢幕護眼、on-device 隱私、跨世代文化傳承；扣回「臺灣下一個十年」。

## 4. 現場風險與備援

| 風險 | 備援 |
| :-- | :-- |
| 硬體當機/開機失敗 | 立即切**預錄 60–90s demo 影片** |
| 現場網路不穩 | 鏡頭 1–2 本就離線，不受影響；鏡頭 4 用預抓的儀表板快照 |
| ASR 誤判/噪音 | 用固定示範語句、彩排時調降噪；必要時 push-to-talk |
| Hermes/Bedrock 延遲 | 儀表板改讀預生成的診斷結果（非同步本就允許） |
| 時間超時 | 鏡頭 2、4 為必留；3、5 可壓縮 |
